# Code Katas

This is a collection of code katas with solutions in Java.

## Kata 01 

Return the longest contiguous substring of 2 distinct characters from an input string.

Example 01
```
input: abbaacab
output : abbaa
```

Example 02
```
input: abcefabbabaabefghghfa 
return: abbabaab
```

Example 03
```
input: aabceddddcdccecabceftg 
return: ddddcdcc
```

Example 04
```
input: acbabbcbca 
return : bbcbc
```
